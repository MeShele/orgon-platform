# API

The canonical API spec is the live OpenAPI schema generated by FastAPI.

| Surface | URL |
|---|---|
| Swagger UI | https://orgon-preview-api.asystem.kg/api/docs |
| ReDoc      | https://orgon-preview-api.asystem.kg/api/redoc |
| OpenAPI JSON | https://orgon-preview-api.asystem.kg/api/openapi.json |

Production equivalents: replace `orgon-preview-api` with `orgon-api`.

The OpenAPI schema is the **source of truth**. Don't write request/response
shapes in markdown — they will go stale. If you need to publish a contract
externally, snapshot the JSON.

---

## Authentication

All authenticated endpoints expect `Authorization: Bearer <jwt>`.

```bash
# Obtain a token
curl -X POST https://orgon-preview-api.asystem.kg/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email": "demo-admin@orgon.io", "password": "demo2026"}'

# Use it
curl https://orgon-preview-api.asystem.kg/api/dashboard/stats \
  -H "Authorization: Bearer <access_token>"
```

Token lifetime: access 1h, refresh 7d. Refresh via `POST /api/auth/refresh`.

Login is rate-limited to **5 attempts/min per IP** (`/api/auth/login`,
`/api/auth/verify-2fa`, `/api/auth/reset-password`,
`/api/auth/reset-password/confirm`). General API limit: 100 req/min/IP.

---

## Multi-tenancy

Every authenticated request must include `X-Organization-ID: <uuid>` to scope
the response to one organization. The middleware verifies the user belongs
to that org before forwarding.

---

## Partner API (B2B)

`/api/v1/partner/*` endpoints accept `X-Partner-API-Key: <key>` instead of a
Bearer token. Partner keys are issued via the Coolify-side admin tooling
(not exposed in UI yet); they are tier-rate-limited (`PartnerRateLimit`
middleware).

---

## Webhooks

Outbound: `WebhookService` posts JSON to a partner-registered URL with
`X-ORGON-Signature: t=<unix>,v1=<hmac-sha256>` header. Verify on the
partner side before processing.

Inbound: `POST /api/webhooks/safina/callback` for Safina-side events.
HMAC verification using `SAFINA_WEBHOOK_SECRET` `(planned — not yet wired)`.
