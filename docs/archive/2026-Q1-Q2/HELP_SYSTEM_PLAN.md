#### Wallets (Кошельки)
**Приоритет: ВЫСОКИЙ**

| Элемент | Описание | Пример | Советы |
|---------|----------|--------|--------|
| ✅ Кнопка "Экспорт" | Сохранить список в CSV | "Экспорт для бухгалтерии" | • Содержит: имя, адрес, баланс<br>• Открывается в Excel |
| Кнопка "Создать" | Новый мультиподписный кошелек | "Создать кошелек 2/3 для проекта" | • Минимум 2 участника<br>• Порог ≥ 2 подписи |
| Колонка "Сеть" | Блокчейн где создан кошелек | "Ethereum (ETH)" | • Определяет комиссии<br>• Нельзя изменить после создания |
| Колонка "Порог" | Минимум подписей для транзакции | "2/3 = нужны 2 из 3 подписей" | • Безопасность vs удобство<br>• 100% = все должны подписать |
| Иконка "Избранное" | Быстрый доступ к важным | "Отметить для главной страницы" | • Макс 5 избранных<br>• Показываются первыми |

#### Transactions (Транзакции)
**Приоритет: ВЫСОКИЙ**

| Элемент | Описание | Пример | Советы |
|---------|----------|--------|--------|
| Фильтр "Статус" | Отбор по состоянию | "Показать только ожидающие" | • Pending = нужны подписи<br>• Completed = в блокчейне |
| Фильтр "Сеть" | Отбор по блокчейну | "Только Ethereum" | • Помогает найти дорогие транзакции (газ) |

| Колонка "Комиссия" | Стоимость выполнения | "0.003 ETH (~$7)" | • Зависит от загрузки сети<br>• Можно ускорить за доп. плату |
| Badge статуса | Визуальный индикатор | "🟡 Ожидает → 🟢 Подтверждена" | • Желтый = в процессе<br>• Зеленый = успех<br>• Красный = ошибка |

**Приоритет: ВЫСОКИЙ**

| Элемент | Описание | Пример | Советы |
|---------|----------|--------|--------|

| Счетчик подписей | Прогресс одобрения | "2/3 подписали (нужна еще 1)" | • Зеленый = готово<br>• Серый = ждем остальных |
