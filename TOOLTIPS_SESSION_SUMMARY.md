## 💡 Ключевые улучшения

### 1. Расширенный компонент HelpTooltip
```typescript
interface HelpTooltipProps {
  text: string;          // Основное описание
  example?: string;      // Пример использования
  tips?: string[];       // Массив советов (3-5 штук)
  diagram?: string;      // Опциональная диаграмма
}
```

### 2. Структура контента
Каждый тултип содержит:
- **text** - Краткое описание (1-2 предложения)
- **example** - Реальный пример использования
- **tips** - 3-5 практических советов

### 3. Диаграммы
Добавлены Mermaid-диаграммы для сложных концепций:
- `tx_lifecycle` - Жизненный цикл транзакции
- `multisig_flow` - Процесс мультиподписи
- `ec_signing` - Процесс подписания EC-ключом
- `token_format` - Формат идентификатора токена
