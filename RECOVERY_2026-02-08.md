## Solution Applied

### 1. Created `backend/dependencies.py`

Вынес все `get_*_service()` функции из `main.py` в отдельный модуль для разрыва циркулярных зависимостей:

```python
# backend/dependencies.py
def get_wallet_service(request: Request):
    """Get WalletService from app state."""
    return request.app.state.wallet_service

# ... 12 других сервисов
```

### 2. Fixed imports in all route files

```bash
# Заменил во всех файлах backend/api/routes_*.py:
from backend.main import get_*_service
# на:
from backend.dependencies import get_*_service
```

## Lessons Learned

### Anti-Patterns Discovered

1. **Circular imports in FastAPI:**
   - ❌ `routes.py` imports from `main.py` → `main.py` imports `routes.py`
   - ✅ Solution: Separate `dependencies.py` module

2. **Automatic code refactoring risks:**
   - ❌ Regex-based sed/awk replacements can corrupt files
   - ✅ Solution: Git restore + manual fixes

3. **Adding features mid-session:**
   - ❌ Prometheus metrics added without testing
   - ✅ Solution: Incremental testing, Git checkpoints

### Best Practices

1. **Git commits before major changes:**
   ```bash
   git commit -m "Before adding metrics"
   ```

2. **Dependency injection pattern:**
   ```python
   # dependencies.py (no imports from main.py)
   def get_service(request: Request):
       return request.app.state.service
   ```

3. **Always test imports:**
   ```bash
   python3 -c "from backend.main import app; print('OK')"
   ```
