# 07. UX Рекомендации и подсказки — ORGON Platform

> **Принцип:** Каждая страница должна быть понятна пользователю без чтения документации.  
> **Языки:** ru / en / ky  
> **Аудитория:** CompanyAdmin, CompanyOperator, EndUser  
> **Дата:** Февраль 2026

---

## Общие принципы UX

### 1. Каждая страница содержит:
- **Заголовок** с иконкой и кратким описанием
- **Onboarding tip** (показывается 1 раз для новых пользователей)
- **Empty state** с иллюстрацией и CTA-кнопкой
- **Tooltips** на сложных элементах (иконка ❓)
- **Confirmation dialog** для необратимых действий
- **Success/Error feedback** после каждого действия

### 2. Состояния элементов
```
Empty → Loading → Data → Error → Refresh
                    ↓
              Action → Confirming → Processing → Success/Failed
```

### 3. Цветовая кодировка статусов
| Статус | Цвет | Пример |
|--------|------|--------|
| Active / Success | 🟢 Green | Транзакция подтверждена |
| Pending / Processing | 🟡 Yellow/Orange | Ожидает подписи |
| Warning / Review | 🟠 Orange | Средний риск |
| Error / Blocked | 🔴 Red | Транзакция отклонена |
| Info / Neutral | 🔵 Blue | Информационное сообщение |
| Disabled / Inactive | ⚪ Gray | Неактивный кошелёк |

## Страницы: Подробные UX рекомендации

### 🏠 Landing Page (`/`)

**Цель:** Привлечь потенциальных клиентов (обменники, биржи, банки).

**Onboarding tip:** Не применимо (публичная страница).

**Ключевые элементы:**
- Hero секция: "Криптокастоди для вашего бизнеса" + CTA "Начать бесплатно"

- Сравнение тарифов (A/B/C) с кнопкой "Выбрать"
- Секция "Для кого": обменники, биржи, банки, финтех
- FAQ: 5-7 вопросов
- Footer: контакты, ссылки, соцсети

```

🇬🇧 "Over 40 companies in Kyrgyzstan can already use the platform"
```

**Onboarding tip:**
```
🇷🇺 "📋 Здесь отображаются все транзакции вашей организации. 
     Используйте фильтры для быстрого поиска."

     Use filters for quick search."
```

**Empty state:**
```
🇷🇺 "Пока нет транзакций"
    "Создайте кошелёк и отправьте первую транзакцию"
    CTA: [Новая транзакция]
```

**Статусы транзакций:**
| Статус | Иконка | Описание RU | Описание EN |
|--------|--------|------------|-------------|
| Created | 🔵 | Создана, ожидает подписей | Created, awaiting signatures |
| Pending signatures | 🟡 | Ожидает подписей (2/3) | Awaiting signatures (2/3) |
| Signed | 🟠 | Подписана, отправляется в сеть | Signed, broadcasting to network |
| Confirming | 🟡 | В блокчейне, ожидает подтверждений | In blockchain, awaiting confirmations |
| Confirmed | 🟢 | Подтверждена | Confirmed |
| Failed | 🔴 | Ошибка отправки | Failed to send |
| Rejected | ⚫ | Отклонена подписантом | Rejected by signer |

**Фильтры:**
- Период (сегодня / неделя / месяц / произвольный)
- Статус (all / pending / confirmed / failed)
- Направление (incoming / outgoing / all)
- Кошелёк
- Сумма (от / до)

### 1. Копирование адресов
